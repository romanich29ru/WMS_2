# Формат XLS для импорта (актуальный)

Этот файл описывает формат `Занятость ячеек.xlsx`, который используется в `WMS_Пусто_занято.html`.

## Лист
- Ожидается первый лист (например, `WMS`).

## Обязательные колонки
1. `Ячейка` — название ячейки (адрес).
2. `Статус` — статус ячейки.

## Поддерживаемые колонки (дополнительно)
- `Дивизион` — отображает, какой склад.
- `Зона` — зона на складе.
- `Шк ячейки` — штрихкод ячейки.
- `Артикул` — артикул товара.
- `Название товара` — название товара.
- `Дата размещения` — дата размещения товара/ячейки.

## Пример заголовков (как в файле)
```
Дивизион | Зона | Шк ячейки | Ячейка | Статус | Артикул | Название товара | Дата размещения
```

## Пример строки
```
Склад-1 | Z1 | 1234567890123 | A01-01-15-A | занято | SKU-001 | Винт М5 | 2026-02-08
```

## Примечания
- Заголовки могут содержать лишние пробелы — они игнорируются.
- `Статус` распознаётся по ключевым словам: `занято/occupied` и `свободно/empty`.
- Если в XLS несколько строк на одну ячейку с разными артикулами, система агрегирует их и ставит флаг конфликта.
